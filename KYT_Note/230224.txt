머티리얼의 노드)


머티리얼의 베이스컬러는 0~1사이의 값을 사용한다.



constants : 머티리얼 인스턴스로 바꿀수 없는 값들
	-> 파라미터 종류가 아닌 멤버는 인스턴스로 변경할 수 없다.

	-> Constant3Vector : RGB값을 사용(3가지 feature)
	-> 색상을 설정 후 좌측 상단의 적용버튼을 눌러야 적용됨
	

Metalic : Constant 하나를 사용
	-> 스칼라값 하나를 사용 : 0~1 사이의 값
	-> 얼마나 금속같은 재질인지를 설정


스페큘러 : 비금속표면의 하이라이터

러프니스 : 빛 반사; 0이면 주변의 빛을 반사함

이미시브 컬러 : 자체적으로 빛을 발산하는속성
	-> 베이스컬러를 가져와서 Multiply 를 함
	-> 얼마나 멀티플라이한 값의 크기에 따라 빛을 얼마나 발광할 것인지 설정가능

머티리얼 파라미터 컬렉션
	-> 스칼라 파라미터와 벡터파라미터가 있다.
	-> 미리 값, 색상등을 설정해서 이를 가져다 사용할 수 있다.

	-> ex.) 라인컬러를 결정해주는 함수 등을 정의해서 사용할 수 있다.

	-> 최종 머티리얼을 결정하는 함수를 정의한다.
	-> 함수의 인자를 설정해서, 이 함수의 호출자가 인자에 값을 넘겨줄 수 있다.
