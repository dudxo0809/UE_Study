레벨은 2가지로 나뉜다.

	-> 오픈월드 & 베이직
	

오픈월드 : 언리얼 5부터 나옴
	-> 월드파티션 기능 : 알아서 공간을 나누고 최적화 함
	-> 기본적으로 월드파티션 하나를 자동으로 만듬
		-> 셀 단위로 쪼개 일부를 로딩함
	-> 화면에 보이는 부분만 처리함
		

베이직 : 언리얼 5이전에는 베이직밖에 없었음
	-> 베이직에서 오픈월드를 만드려면 월드 스트리밍을 사용해야했다.

	-> 랜드스케이프 머티리얼을 설정한다.



랜드스케이프 전용 레이어 머티리얼이 존재

	-> 쉬프트+2 : 랜드스케이프 모드에서
	-> 페인트


Material Landscape
	-> Landscape 에서 레이어 블렌드
	밑의 레이어는 위의 레이어를 덮어서 출력할 수 있다.


	블렌드 타입 : 가중치 타입, 알파타입, 높이타입(산 같은 지형에서 높이에 따른 텍스처를 입히는용도)
			-> 기본적으로는 weight blend를 사용(가중치)
			-> 페인트를 이용해 원하는 레이어의 가충치를 설정할 수 있다.


	Layer Coord : 랜드스케이프용 UV좌표



	랜드스케이프 레이어마다 피직스 머티리얼을 다로 설정한다.
	-> 돌, 흙, 풀을 밟을때의 소리 등

보통 월드포지션 오프셋을 이용해 파도가 치는 시뮬레이션을 돌린다.

물을 생성
	-> 물이 있는 바닥에 water Layer를 깔아놔야 더 이쁘게 나옴
	-> 머티링러하나만 만들면 끝
	-> 반투명이므로 Transluscent로 설정하고, 양면으로 설정
	-> 굴절 처리가 필요

스크린스페이스 리플렉션 True로 설정
Translucency의 라이팅모드를 Surface Translucency Volume으로 둠
	-> 메탈릭 등의 속성이 살아남
스태틱라이팅과 사용됨 True로 설정
리플렉션모드 : 일정한 굴졀률을 가질지, 말지 설정
		-> Pixel Normal Offset으로 설정함

플로이드 이펙트 : 실제로 물을 채우는 효과 : 나이아가라로 해야함


깊이에따라 반투명도가 달라지게해야함
	-> Plane 과 바닥의 거리(깊이)를 구해서 설정
	-> Depth Fade사용 : Opacity에 할당
	-> Mask : 3feature중 2개만 사용


	흐르는 듯한 효과를 위해 Panner를 사용
		-> 2축에대한 속도를 생성
